# UI админ панели
Реализация UI админ панели на next.js
# Установка
1. Клонируйте репозиторий:

    ```sh
    git clone https://github.com/bosonhiiggs/MT-Client.git](https://github.com/ArtRetro/Fittin-test.git
    ```

2. Перейдите в директорию проекта:

    ```sh
    cd admin-panel
    ```

3. Установите зависимости:

    ```sh
    npm install
    # или
    yarn install
    # или
    pnpm install
    ```

  4. Запустите проект:

     ```sh
     npm run dev
     # или
     yarn dev
     # или
     pnpm dev
     ```
