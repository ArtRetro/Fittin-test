# Установка
1. Клонируйте репозиторий:

    ```sh
    git clone https://github.com/bosonhiiggs/MT-Client.git](https://github.com/ArtRetro/Fittin-test.git
    ```

2. Перейдите в директорию проекта:

    ```sh
    cd Fittin_test
    ```

3. Установите зависимости:

    ```sh
    flutter pub get
    ```

4. Запустите приложение:

    ```sh
    flutter run
    ```
